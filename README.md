## 📌 Overview

**CloudVault** is a self-hosted personal cloud storage platform that allows users to securely upload, store, manage, and share their files from anywhere.

The project provides a private alternative to cloud services like Google Drive and Dropbox by giving users complete control over their data, storage, and security.

# ⚙️ Installation

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/cloudvault.git
```

## 2. Enter Project Directory

```bash
cd cloudvault
```

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

## 4. Configure Database

Update database settings:

